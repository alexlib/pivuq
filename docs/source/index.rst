Welcome to pivuq's documentation!
=================================

**PIV-UQ** (`pivuq`) is Python library for PIV Uncertainty Quantification. Primary aim is to implement UQ algorithms for PIV techniques. Future goals include possible extensions to other domains including but not limited to optical flow and BOS.
